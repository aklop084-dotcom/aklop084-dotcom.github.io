Билет 2. Матрицы. Сложение матриц, умножение матрицы на число, свойства этих операций. Понятие линейного пространства.

Матрицы и операции над ними

Матрицы представляют собой удобный способ записи и обработки больших объемов числовой информации, особенно в контексте систем линейных уравнений. Нам нужно уметь выполнять арифметические операции с матрицами для описания преобразований в многомерных пространствах или решения сложных физических задач.

1. Определение матрицы

Словами: матрицу можно описать как прямоугольную таблицу, заполненную числами, которые расположены в строках и столбцах. Каждое число в этой таблице называется элементом матрицы.

Строгое математическое определение: матрицей размера m×n называется совокупность mn чисел, расположенных в виде таблицы из m строк и n столбцов:

A = ⎡ a₁₁ a₁₂ … a₁ₙ ⎤
    ⎢ a₂₁ a₂₂ … a₂ₙ ⎥
    ⎢ … … … … ⎥
    ⎣ aₘ₁ aₘ₂ … aₘₙ ⎦

Элемент, стоящий на пересечении i-й строки и j-го столбца, обозначается aᵢⱼ. Если число строк равно числу столбцов (m = n), матрица называется квадратной, а число n — ее порядком.

2. Сложение матриц

Словами: операция сложения описывается предельно просто: чтобы сложить две матрицы, нужно сложить их элементы, стоящие на одинаковых позициях. Важно помнить, что складывать можно только матрицы одинакового размера.

Строгое математическое определение: пусть даны две матрицы A = (aᵢⱼ) и B = (bᵢⱼ) размера m×n. Суммой матриц A и B называется матрица C = A + B того же размера m×n, элементы которой вычисляются по формуле:

cᵢⱼ = aᵢⱼ + bᵢⱼ     (i = 1, …, m; j = 1, …, n)

Пример:

A = ⎡ 1   2 ⎤
    ⎣ 3   4 ⎦
,
B = ⎡ 5   6 ⎤
    ⎣ 7   8 ⎦
, то
A + B = ⎡ 1+5   2+6 ⎤ = ⎡ 6   8 ⎤
          ⎣ 3+7   4+8 ⎦     ⎣ 10   12 ⎦

3. Умножение матрицы на число

Словами: это означает, что каждый элемент матрицы нужно умножить на заданное число (скаляр).

Математически: произведением матрицы A = (aᵢⱼ) на число α называется матрица C = αA, элементы которой определяются равенством:

cᵢⱼ = α·aᵢⱼ     (i = 1, …, m; j = 1, …, n)

4. Свойства операций сложения и умножения на число

Эти операции обладают рядом свойств, которые позволяют оперировать матрицами почти так же, как обычными числами. Нам нужно знать эти свойства для того, чтобы уверенно преобразовывать матричные выражения, подобно тому как мы делаем это в обычной алгебре.

Пусть A, B, C — матрицы одного размера, а α, β — произвольные числа.

❶ A + B = B + A
(коммутативность сложения)
❷ (A + B) + C = A + (B + C)
(ассоциативность сложения)
❸ ∃ O: A + O = A
(существование нулевой матрицы)
❹ ∃ (-A): A + (-A) = O
(существование противоположной матрицы)
❺ α(βA) = (αβ)A
(ассоциативность умножения на числа)
❻ α(A + B) = αA + αB
(дистрибутивность относительно сложения матриц)
❼ (α + β)A = αA + βA
(дистрибутивность относительно сложения чисел)
❽ 1·A = A
(умножение на единицу)

Понятие линейного пространства

Нам нужно понятие линейного пространства для создания единого математического аппарата, который подходит и для векторов, и для матриц, и для функций. Линейное пространство — это обобщение понятия векторов на плоскости или в пространстве.

1. Определение словами

Линейное (или векторное) пространство — это множество объектов любой природы (их называют векторами), для которых определены две операции: сложение друг с другом и умножение на число. Эти операции должны подчиняться восьми строгим правилам (аксиомам), которые гарантируют «правильное» поведение объектов внутри этого множества.

2. Строгое математическое определение

Множество L называется линейным пространством, а его элементы — векторами, если:

  • Задан закон (операция сложения), по которому любым двум элементам x, y ∈ L сопоставляется элемент x + y ∈ L.
  • Задан закон (операция умножения на число), по которому любому элементу x ∈ L и любому числу α сопоставляется элемент αx ∈ L.

Для любых элементов x, y, z ∈ L и любых чисел α, β должны выполняться следующие аксиомы:

  • Аксиома 1. x + y = y + x (коммутативность сложения)
  • Аксиома 2. (x + y) + z = x + (y + z) (ассоциативность сложения)
  • Аксиома 3. Существует нулевой вектор 0, такой что x + 0 = x для любого x
  • Аксиома 4. Для любого x существует противоположный элемент −x, такой что x + (−x) = 0
  • Аксиома 5. α(x + y) = αx + αy
  • Аксиома 6. (α + β)x = αx + βx
  • Аксиома 7. α(βx) = (αβ)x
  • Аксиома 8. 1·x = x

3. Матрицы как пример линейного пространства

Поскольку операции над матрицами (сложение и умножение на число), разобранные выше, удовлетворяют всем восьми аксиомам, множество всех матриц размера m×n является классическим примером линейного пространства. Это позволяет нам применять к матрицам все методы линейной алгебры: находить базис, размерность и исследовать их линейную зависимость.
