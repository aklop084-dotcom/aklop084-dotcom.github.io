Билет 9. Скалярное произведение двух векторов. Выражение ортогональной проекции одного вектора на другой с помощью скалярного произведения. Линейность операции ортогонального проектирования. Скалярное произведение двух векторов

Скалярным произведением двух векторов называется число (скаляр), которое получается в результате умножения длин этих векторов на косинус угла, образованного между ними. Если хотя бы один из векторов является нулевым, то угол не определен, а скалярное произведение по определению принимается равным нулю. Нам нужно это понятие для решения фундаментальных задач геометрии и алгебры, например, для вычисления длин векторов и определения взаимного расположения объектов в пространстве.

📌 Определение скалярного произведения

На строгом математическом языке определение формулируется следующим образом: Пусть даны два вектора a и b. Скалярное произведение обозначается (a, b) или a·b. Оно определяется формулой:

(a, b) = |a|·|b|·cos φ

где |a| и |b| — длины (модули) векторов, а φ — угол между ними (0 ≤ φ ≤ π).

Ортогональная проекция вектора

Ортогональной проекцией вектора на прямую (или на другой вектор) называется такой вектор, который лежит на этой прямой, а его начало и конец являются основаниями перпендикуляров, опущенных из начала и конца исходного вектора на данную прямую.

📌 Пошаговый метод нахождения проекции
  • Шаг 1. Найти скалярное произведение векторов a и b. Это число покажет нам «вклад» одного вектора в направление другого.
  • Шаг 2. Вычислить квадрат длины вектора b, на который мы проектируем. Нам нужно это значение для нормировки результата.
  • Шаг 3. Составить коэффициент, разделив скалярное произведение на квадрат длины вектора b.
  • Шаг 4. Умножить полученный коэффициент на сам вектор b. Полученный результат и будет искомым вектором проекции.
📌 Выражение для векторной проекции
Прᵦ a = ((a, b) / |b|²) · b

Также существует понятие скалярной проекции (координаты проекции), которая обозначается прᵦ a и равна:

прᵦ a = |a|·cos φ = (a, b) / |b|

Линейность операции ортогонального проектирования

Операция проектирования обладает свойством линейности. Это означает, что проекция суммы векторов равна сумме их проекций, а проекция вектора, умноженного на число, равна произведению этого числа на проекцию исходного вектора.

📐 Математическая запись: для любого проектирования φ:
φ(x + y) = φ(x) + φ(y)     для любых векторов x, y
φ(λx) = λ·φ(x)     для любого числа λ

Это свойство крайне важно, так как оно позволяет рассматривать проектирование как линейное отображение (или оператор) в векторном пространстве.

Свойства скалярного произведения

Скалярное произведение обладает рядом свойств, которые позволяют оперировать с ним аналогично обычным числам:

❶ Коммутативность (симметричность)
Результат не зависит от порядка векторов.
(a, b) = (b, a)
❷ Линейность по первому аргументу
Скалярное произведение суммы векторов на третий вектор равно сумме их скалярных произведений, а числовой множитель можно выносить за знак произведения.
(αa + βb, c) = α(a, c) + β(b, c)
❸ Положительная определенность
Скалярный квадрат вектора (произведение вектора на самого себя) всегда не меньше нуля и равен нулю только для нулевого вектора.
(a, a) = |a|² ≥ 0
❹ Условие ортогональности
Скалярное произведение двух ненулевых векторов равно нулю тогда и только тогда, когда эти векторы перпендикулярны.
(a, b) = 0 ⇔ a ⟂ b

Вычисление в координатах и ортонормированный базис

Ортонормированным базисом называется такая система базисных векторов, в которой все векторы имеют единичную длину и попарно перпендикулярны (ортогональны) друг другу. Нам нужно использовать именно такой базис для упрощения вычислений, чтобы формула скалярного произведения приняла максимально простой вид.

📌 Скалярное произведение в ортонормированном базисе

Если векторы заданы своими координатами в ортонормированном базисе {e₁, e₂, e₃}, то есть a = α₁e₁ + α₂e₂ + α₃e₃ и b = β₁e₁ + β₂e₂ + β₃e₃, то их скалярное произведение вычисляется как сумма произведений соответствующих координат:

(a, b) = α₁β₁ + α₂β₂ + α₃β₃
📌 Разложение вектора по ортонормированному базису

Коэффициенты разложения (координаты) — это в точности скалярные произведения самого вектора на соответствующие векторы базиса:

a = (a, e₁)e₁ + (a, e₂)e₂ + (a, e₃)e₃

Матрица Грама

В случае, если базис вектора не является ортонормированным, формула скалярного произведения усложняется. Для учета метрики такого пространства используется матрица Грама.

📌 Определение матрицы Грама

Матрицей Грама системы векторов {e₁, …, eₙ} называется квадратная матрица, элементами которой являются скалярные произведения векторов этой системы.

gᵢⱼ = (eᵢ, eⱼ)
G = ⎡ (e₁,e₁) (e₁,e₂) … (e₁,eₙ) ⎤
      ⎢ (e₂,e₁) (e₂,e₂) … (e₂,eₙ) ⎥
      ⎢ … … … … ⎥
      ⎣ (eₙ,e₁) (eₙ,e₂) … (eₙ,eₙ) ⎦
📌 Скалярное произведение через матрицу Грама

С помощью этой матрицы скалярное произведение векторов x = Σ ξᵢ eᵢ и y = Σ ηⱼ eⱼ в произвольном базисе записывается в матричном виде:

(x, y) = Σᵢ,ⱼ gᵢⱼ ξᵢ ηⱼ = ξᵀ G η

где ξ и η — столбцы координат векторов. Если базис ортонормирован, то матрица Грама становится единичной матрицей E.
