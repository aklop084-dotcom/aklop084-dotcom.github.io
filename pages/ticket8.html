Векторы в геометрии и линейном пространстве

Понятие вектора является фундаментальным в математике и физике. Нам необходимо оперировать векторами для описания физических сил, скоростей или создания математических моделей в многомерных пространствах.

1. Определение вектора и линейные операции

Словами: В геометрии вектором называется направленный отрезок, то есть отрезок, для которого указано, какая из его граничных точек является началом, а какая — концом. Мы рассматриваем так называемые «свободные векторы»: два вектора считаются равными, если они коллинеарны (лежат на параллельных прямых или одной прямой), имеют одинаковое направление и равную длину.

Строгим математическим языком: Вектор обозначается как AB (где A — начало, B — конец) или одной строчной буквой a. Длина вектора обозначается |a|. Над векторами определены две основные операции:

📐 Сложение векторов

Суммой векторов AB и BC называется вектор AC (правило треугольника).

📐 Умножение вектора на число

Произведением вектора a на вещественное число α называется вектор b = αa, такой что:

  • его длина |b| = |α|·|a|;
  • он коллинеарен исходному вектору a;
  • его направление совпадает с направлением a, если α > 0, и противоположно ему, если α < 0.

2. Множество геометрических векторов как линейное пространство

Нам нужно знать свойства операций над векторами для доказательства того, что совокупность всех векторов образует единую алгебраическую структуру — линейное пространство.

Словами: Линейное (или векторное) пространство — это множество объектов, которые можно складывать друг с другом и умножать на числа, при этом результат операции не должен выходить за пределы этого множества, а сами операции должны подчиняться восьми правилам (аксиомам).

Строгим математическим языком: Множество всех векторов на плоскости (обозначается ℝ²) или в пространстве (ℝ³) является вещественным линейным пространством, так как для любых векторов x, y, z и чисел α, β выполняются следующие аксиомы:

❶ x + y = y + x
(коммутативность)
❷ (x + y) + z = x + (y + z)
(ассоциативность)
❸ ∃ 0: x + 0 = x
(существование нулевого вектора)
❹ ∃ (-x): x + (-x) = 0
(существование противоположного)
❺ 1·x = x
(умножение на единицу)
❻ α(βx) = (αβ)x
(ассоциативность умножения на числа)
❼ (α + β)x = αx + βx
(дистрибутивность относительно сложения чисел)
❽ α(x + y) = αx + αy
(дистрибутивность относительно сложения векторов)

3. Линейная зависимость и независимость векторов

Словами: Векторы называются линейно зависимыми, если хотя бы один из них можно представить как комбинацию других (выразить через остальные с помощью сложения и умножения на числа). Если же ни один вектор в наборе нельзя выразить через другие, то они называются линейно независимыми.

📌 Определение линейной независимости

Система векторов a₁, a₂, …, aₖ называется линейно независимой, если равенство

α₁a₁ + α₂a₂ + ⋯ + αₖaₖ = 0

выполняется тогда и только тогда, когда все коэффициенты αᵢ одновременно равны нулю.

📌 Определение линейной зависимости

Если же существуют такие числа αᵢ, среди которых есть хотя бы одно ненулевое, что эта сумма (линейная комбинация) равна нулевому вектору, то система векторов называется линейно зависимой.

4. Базис и координаты вектора

Нам нужно понятие базиса для перехода от геометрических чертежей к точным числовым расчетам.

Словами: Базис — это такой минимальный набор линейно независимых векторов, с помощью которых можно однозначно «собрать» любой другой вектор данного пространства. Числа, которые показывают, в каких пропорциях нужно взять базисные векторы для получения искомого, называются координатами вектора в этом базисе.

📌 Строгое определение базиса

Базисом в линейном пространстве называется упорядоченная система векторов e₁, …, eₙ, если:

  • эта система линейно независима;
  • любой вектор a данного пространства может быть представлен в виде линейной комбинации:
a = x₁e₁ + x₂e₂ + ⋯ + xₙeₙ

Коэффициенты x₁, x₂, …, xₙ называются координатами (компонентами) вектора a в данном базисе. Для конкретного базиса набор координат вектора единственен.

5. Базисы на прямой, плоскости и в пространстве

Выбор базиса зависит от размерности пространства, в котором мы работаем:

📐 На прямой (1D): Базисом является любой один ненулевой вектор. Любой вектор на этой прямой будет ему коллинеарен.
📐 На плоскости (2D): Базисом является любая упорядоченная пара неколлинеарных векторов. Любые три вектора на плоскости всегда будут линейно зависимы (компланарны).
📐 В пространстве (3D): Базисом является любая упорядоченная тройка некомпланарных векторов (не лежащих в одной плоскости). Любые четыре вектора в трехмерном пространстве линейно зависимы.

6. Запись операций над векторами в координатах

Координатная форма записи позволяет заменить геометрические построения арифметикой. Пусть в базисе e₁, …, eₙ даны векторы a(x₁, …, xₙ) и b(y₁, …, yₙ).

📐 Сложение векторов

Координаты суммы равны суммам соответствующих координат:

a + b = (x₁ + y₁, …, xₙ + yₙ)
📐 Умножение вектора на число

Каждая координата умножается на это число:

αa = (αx₁, …, αxₙ)

7. Условия линейной независимости в координатах

Нам необходимо уметь проверять векторы на независимость аналитически.

📌 Для двух векторов на плоскости

Словами: векторы независимы, если они не пропорциональны (не коллинеарны).

Математически: векторы a(x₁, y₁) и b(x₂, y₂) линейно независимы тогда и только тогда, когда определитель, составленный из их координат, не равен нулю:

| x₁ x₂ | ≠ 0
| y₁ y₂ |

Если же определитель равен нулю, векторы коллинеарны и зависимы.

📌 Для трех векторов в пространстве

Словами: векторы независимы, если они не компланарны (не лежат в одной плоскости).

Математически: условием линейной независимости векторов a(x₁, y₁, z₁), b(x₂, y₂, z₂), c(x₃, y₃, z₃) является отличие от нуля определителя третьего порядка, составленного из их координат:

| x₁ y₁ z₁ |
| x₂ y₂ z₂ | ≠ 0
| x₃ y₃ z₃ |

Это значение также соответствует (с точностью до знака) ориентированному объему параллелепипеда, построенного на этих векторах. Если объем не равен нулю, векторы образуют базис в пространстве.
