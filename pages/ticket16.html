Линейное пространство

📌 Определение и аксиомы

Линейным (или векторным) пространством называется множество элементов любой природы (их называют векторами), в котором определены операции сложения и умножения на число, не выводящие за пределы этого множества. Нам нужно строгое понятие линейного пространства для построения единой теории, описывающей свойства функций, матриц и геометрических векторов.

Строгим математическим языком: Множество L называется линейным пространством над полем K (обычно ℝ или ℂ), если заданы операции сложения x + y и умножения на число αx, удовлетворяющие восьми аксиомам:

  • Аксиома 1. x + y = y + x (коммутативность сложения).
  • Аксиома 2. (x + y) + z = x + (y + z) (ассоциативность сложения).
  • Аксиома 3. Существует нулевой вектор 0, такой что x + 0 = x для любого x.
  • Аксиома 4. Для каждого x существует противоположный вектор -x, такой что x + (-x) = 0.
  • Аксиома 5. α(x + y) = αx + αy (дистрибутивность относительно сложения векторов).
  • Аксиома 6. (α + β)x = αx + βx (дистрибутивность относительно сложения чисел).
  • Аксиома 7. α(βx) = (αβ)x (ассоциативность умножения на число).
  • Аксиома 8. 1·x = x (умножение на единицу).

Примеры линейных пространств

ℝⁿ
Арифметическое пространство: множество всех упорядоченных наборов из n вещественных чисел.
M(m×n)
Пространство матриц: множество всех матриц размера m×n с вещественными элементами.
Pₙ(x)
Пространство многочленов: множество всех многочленов от одной переменной, степень которых не превышает n.
C[a,b]
Пространство функций: множество всех непрерывных функций на отрезке [a, b].

Линейная зависимость и независимость векторов

Это понятие позволяет отличить «уникальные» векторы от тех, которые можно собрать из других. Нам это нужно для того, чтобы понимать размерность пространства и не хранить лишнюю информацию.

📌 Определение

Объяснение словами: Набор векторов называется линейно зависимым, если хотя бы один из них можно представить как сумму остальных, умноженных на какие-то числа. Если же ни один вектор нельзя «собрать» из других (единственный способ получить нулевой вектор — это умножить все векторы на ноль), то система векторов называется линейно независимой.

Строгий математический язык: Система векторов {x₁, x₂, …, xₖ} называется линейно независимой, если равенство

α₁x₁ + α₂x₂ + ⋯ + αₖxₖ = 0

выполняется тогда и только тогда, когда все коэффициенты αᵢ равны нулю. Если же существует набор коэффициентов, среди которых есть хотя бы один ненулевой, при котором эта сумма равна нулю, то векторы называются линейно зависимыми.

Базис и размерность пространства

📌 Определение базиса

Базисом в линейном пространстве называется упорядоченная система векторов, которая одновременно является линейно независимой и порождает всё пространство (то есть любой вектор пространства можно выразить через них).

📌 Постоянство числа векторов и размерность

Важнейшим свойством конечномерных пространств является то, что все его базисы состоят из одного и того же количества векторов. Это число называется размерностью пространства и обозначается dim L.

Координаты вектора и операции в координатах

Если мы выбрали базис {e₁, …, eₙ}, то любой вектор x можно представить в виде:

x = ξ₁e₁ + ξ₂e₂ + ⋯ + ξₙeₙ

Числа ξ₁, …, ξₙ называются координатами вектора в данном базисе. Нам нужны координаты для перевода абстрактных операций над объектами на язык обычных вычислений с числами.

📌 Запись операций в координатах

Если векторы заданы своими координатами, то:

  • При сложении векторов их соответствующие координаты складываются.
  • При умножении вектора на число все его координаты умножаются на это число.

Изменение базиса и матрица перехода

Когда мы переходим от одного («старого») базиса e к другому («новому») базису e′, возникает необходимость связать их между собой.

📌 Матрица перехода (определение и построение)

Матрицей перехода S от базиса e к базису e′ называется матрица, столбцы которой являются координатами векторов нового базиса, записанными в старом базисе.

Пошаговый метод нахождения матрицы перехода:

  • Берем первый вектор нового базиса e₁′ и находим его разложение по старому базису.
  • Записываем коэффициенты этого разложения в первый столбец матрицы S.
  • Повторяем процедуру для всех остальных векторов нового базиса e₂′, …, eₙ′, заполняя соответствующие столбцы.
📌 Связь координат вектора при изменении базиса

Пусть ξ — столбец координат вектора в старом базисе, а ξ′ — в новом. Тогда они связаны соотношением:

ξ = S·ξ′     или     ξ′ = S⁻¹·ξ

Важно помнить: чтобы найти «новые» координаты через «старые», нужно использовать обратную матрицу перехода S⁻¹.
