Подпространства в линейном пространстве

📌 Определение и основные свойства

Подпространством линейного пространства называется такое его непустое подмножество, которое само является линейным пространством относительно операций, определенных в исходном пространстве. Говоря простыми словами, если мы возьмем часть векторов из большого пространства, то эта часть будет подпространством только в том случае, если при сложении любых векторов из этой части и при умножении их на любые числа мы все равно будем оставаться внутри этой же части векторов. Нам нужно понятие подпространства для классификации решений систем уравнений или изучения структуры линейных отображений.

Строгим математическим языком: Пусть L — линейное пространство. Непустое подмножество L′ ⊂ L называется линейным подпространством, если выполнены следующие условия:

  • Для любых векторов x, y ∈ L′ их сумма x + y также принадлежит L′ (замкнутость относительно сложения).
  • Для любого вектора x ∈ L′ и любого числа α произведение αx также принадлежит L′ (замкнутость относительно умножения на число).

Из этих условий следует, что любое подпространство обязательно содержит нулевой вектор 0, так как 0 = 0·x для любого x ∈ L′.

Способы задания подпространств

Подпространства могут быть заданы двумя принципиально разными способами: «изнутри» и «снаружи».

❶ Задание «изнутри» (Линейная оболочка)
Мы берем конкретный набор векторов и объявляем подпространством совокупность всех их возможных линейных комбинаций. Этот метод часто используется, когда нам нужно построить минимальное пространство, содержащее определенные важные для нас элементы.
❷ Задание «снаружи» (Система уравнений)
Подпространство определяется как совокупность всех векторов, удовлетворяющих некоторому условию, чаще всего — однородной системе линейных уравнений. Это позволяет описывать пространство через ограничения, накладываемые на его элементы.

Понятие ранга конечного набора векторов

Ранг системы векторов — это число, которое показывает, сколько в данной системе есть «настоящих», независимых направлений. Если у нас есть много векторов, некоторые из них могут дублировать информацию других (быть их линейными комбинациями). Ранг позволяет отсечь все лишнее и понять истинную размерность структуры, которую образуют эти векторы.

📌 Определение ранга

Рангом системы векторов {X₁, X₂, …, Xₖ} называется максимальное число линейно независимых векторов в этой системе. Математически это записывается как:

rank{X₁, …, Xₖ} = dim⟨X₁, …, Xₖ⟩

где ⟨…⟩ обозначает линейную оболочку системы.

Линейная оболочка конечного набора векторов и ее размерность

Линейной оболочкой системы векторов называется совокупность всех векторов, которые можно получить, складывая данные векторы с произвольными коэффициентами. Это самое «маленькое» подпространство, в котором лежат все векторы исходного набора.

📌 Определение линейной оболочки

Пусть a₁, a₂, …, aₖ — векторы пространства L. Линейной оболочкой (или подпространством, порожденным этими векторами) называется множество:

⟨a₁, …, aₖ⟩ = {λ₁a₁ + λ₂a₂ + ⋯ + λₖaₖ | λᵢ ∈ ℝ}

Размерность линейной оболочки всегда равна рангу системы векторов, которая ее порождает. Любой максимальный линейно независимый набор векторов из исходной системы будет являться базисом этой линейной оболочки.

📐 Пример

Если в трехмерном пространстве взять два неколлинеарных вектора, их линейная оболочка будет представлять собой плоскость, проходящую через начало координат, а ее размерность будет равна 2.

Подпространство решений однородной системы линейных уравнений

Рассмотрим однородную систему линейных уравнений (ОСЛУ), которую можно записать в матричном виде как AX = 0, где A — матрица коэффициентов размера m×n, а X — столбец неизвестных. Множество всех решений такой системы всегда образует линейное подпространство в пространстве ℝⁿ (или Kⁿ).

📐 Доказательство (словами): Чтобы доказать, что решения ОСЛУ образуют подпространство, нужно проверить два условия. Во-первых, если мы возьмем два решения и сложим их, сумма тоже должна быть решением. Это верно, так как матрица, умноженная на сумму, равна сумме произведений матрицы на каждое решение в отдельности (0 + 0 = 0). Во-вторых, если мы умножим решение на любое число, результат также должен остаться решением, что следует из свойств умножения матрицы на вектор.
📐 Доказательство (строгим языком): Пусть Vₐ = {X ∈ ℝⁿ | AX = 0}.
1. Пусть X⁽¹⁾, X⁽²⁾ ∈ Vₐ. Тогда AX⁽¹⁾ = 0 и AX⁽²⁾ = 0. Следовательно, A(X⁽¹⁾ + X⁽²⁾) = AX⁽¹⁾ + AX⁽²⁾ = 0 + 0 = 0, значит X⁽¹⁾ + X⁽²⁾ ∈ Vₐ.
2. Пусть X ∈ Vₐ и α ∈ ℝ. Тогда AX = 0. Следовательно, A(αX) = α(AX) = α·0 = 0, значит αX ∈ Vₐ.
Так как условия выполнены, Vₐ — подпространство.
📌 Размерность подпространства решений

Размерность этого подпространства решений вычисляется по формуле:

dim Vₐ = n - r

где n — число неизвестных, а r = rank(A) — ранг матрицы системы.

Фундаментальная система решений (ФСР)

Базис в подпространстве решений однородной системы линейных уравнений называется фундаментальной системой решений. Это такой набор решений, через который любое другое решение системы можно выразить единственным образом в виде линейной комбинации.

📌 Метод построения ФСР (пошагово)

Нам нужна ФСР для того, чтобы полностью описать все бесконечное множество решений системы с помощью всего нескольких векторов.

  • Шаг 1. Приведение к ступенчатому виду: С помощью элементарных преобразований строк приводим матрицу системы A к ступенчатому (или улучшенному ступенчатому) виду.
  • Шаг 2. Определение ранга и переменных: Находим ранг r матрицы. Выбираем r главных (базисных) переменных, соответствующих первым ненулевым элементам строк, и n - r свободных переменных.
  • Шаг 3. Выражение главных переменных: Переносим свободные переменные в правую часть уравнений и выражаем через них главные переменные.
  • Шаг 4. Составление векторов ФСР: Чтобы получить ФСР, мы по очереди придаем одной из свободных переменных значение 1, а остальным — 0, и вычисляем соответствующие значения главных переменных.
    • Для первой свободной переменной: (1, 0, …, 0) → получаем вектор X⁽¹⁾.
    • Для второй свободной переменной: (0, 1, …, 0) → получаем вектор X⁽²⁾.
    • И так далее до X⁽ⁿ⁻ʳ⁾.
📌 Общее решение

Теперь любое решение X системы может быть записано как:

X = c₁·X⁽¹⁾ + c₂·X⁽²⁾ + ⋯ + c_{n-r}·X⁽ⁿ⁻ʳ⁾

где cᵢ — произвольные числа.
