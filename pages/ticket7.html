Билет 7. Критерий существования и формула обратной матрицы. Присоединенная матрица. Вычисление обратной матрицы с помощью элементарных преобразований. Два способа решения матричных уравнений вида AX = B, YA = B (где det A ≠ 0): с помощью обратной матрицы и элементарных преобразований.

Обратная матрица: критерий, формулы и методы решения уравнений

Нам необходимо владеть понятием обратной матрицы и уметь её вычислять для того, чтобы решать матричные уравнения так же просто, как мы решаем обычные линейные уравнения с числами. Обратная матрица играет роль «деления» в мире матриц.

1. Критерий существования и формула обратной матрицы

Словами: Матрица называется обратимой, если существует такая матрица, при умножении на которую (как слева, так и справа) получается единичная матрица. Основным условием того, что для квадратной матрицы можно найти обратную, является её «невырожденность». Это означает, что определитель матрицы не должен быть равен нулю.

Строгим математическим языком: Квадратная матрица A порядка n называется обратимой, если существует матрица A⁻¹ такая, что:

AA⁻¹ = A⁻¹A = E

где E — единичная матрица того же порядка.

📌 Теорема (Критерий обратимости)

Матрица A обратима тогда и только тогда, когда она является невырожденной, то есть её определитель det A ≠ 0.

📌 Формула обратной матрицы

Если det A ≠ 0, то обратная матрица вычисляется по формуле:

A⁻¹ = (1/det A) · A^∨

где A^∨ — присоединенная матрица.

2. Присоединенная матрица

Словами: Присоединенная (или союзная) матрица — это специальным образом построенная матрица, состоящая из алгебраических дополнений к элементам исходной матрицы. Важно помнить, что эти дополнения нужно не просто расставить по своим местам, но и обязательно транспонировать полученную таблицу.

Строгим математическим языком: Пусть Aᵢⱼ — алгебраическое дополнение элемента aᵢⱼ матрицы A. Присоединенной матрицей A^∨ (в некоторых источниках обозначается A*) называется транспонированная матрица из алгебраических дополнений:

A^∨ = ⎡ A₁₁ A₂₁ … Aₙ₁ ⎤ ⎢ A₁₂ A₂₂ … Aₙ₂ ⎥ ⎢ … … … … ⎥ ⎣ A₁ₙ A₂ₙ … Aₙₙ ⎦
📐 Свойство присоединенной матрицы: Нам нужно знать свойства присоединенной матрицы для обоснования формулы обратной матрицы через фундаментальное тождество:
AA^∨ = A^∨A = (det A) · E

3. Вычисление обратной матрицы с помощью элементарных преобразований

На практике для матриц большого порядка вычисление через алгебраические дополнения становится слишком трудоемким, поэтому нам нужен более эффективный метод — алгоритм Гаусса–Жордана.

📌 Пошаговое описание метода
  • Шаг 1. Составление расширенной матрицы. Записываем матрицу A, а справа от неё через вертикальную черту приписываем единичную матрицу E того же размера. Получаем блок (A∣E) размера n×2n.
  • Шаг 2. Прямой ход (приведение к ступенчатому виду). С помощью элементарных преобразований строк над всей расширенной матрицей добиваемся того, чтобы на месте матрицы A получилась верхняя треугольная матрица.
  • Шаг 3. Обратный ход (приведение к единичному виду). Продолжаем преобразования строк (делим строки на ведущие элементы и зануляем элементы над главной диагональю), пока левая часть матрицы не превратится в единичную матрицу E.
  • Шаг 4. Результат. Матрица, которая в итоге окажется в правой части (на месте, где изначально была E), и будет искомой обратной матрицей A⁻¹.
(A ∣ E) → (E ∣ A⁻¹)

4. Решение матричных уравнений AX = B и YA = B

Когда нам даны матричные уравнения и известно, что det A ≠ 0, мы можем найти неизвестные матрицы X и Y двумя основными способами.

Способ 1: С помощью обратной матрицы

Для уравнения AX = B: Словами: чтобы «избавиться» от матрицы A, стоящей перед X, нужно умножить всё уравнение на A⁻¹ слева.

A⁻¹(AX) = A⁻¹B → (A⁻¹A)X = A⁻¹B → EX = A⁻¹B → X = A⁻¹B

Для уравнения YA = B: Словами: здесь матрица A стоит после неизвестной Y, поэтому мы должны умножить на A⁻¹ справа.

(YA)A⁻¹ = BA⁻¹ → Y(AA⁻¹) = BA⁻¹ → YE = BA⁻¹ → Y = BA⁻¹
Способ 2: С помощью элементарных преобразований

Нам нужен этот способ для того, чтобы избежать отдельного этапа вычисления обратной матрицы, объединив всё в один вычислительный процесс.

Для уравнения AX = B:

  • Составляем расширенную матрицу из блоков (A ∣ B).
  • С помощью элементарных преобразований строк приводим левую часть A к единичному виду E.
  • В правой части автоматически получится решение X.
(A ∣ B) → (E ∣ X)

Для уравнения YA = B:

  • Здесь удобнее работать со столбцами или использовать транспонирование.
  • Если использовать преобразования столбцов, схема будет:
( B ) → ( Y )
( A ) ( E )

Альтернативно можно перейти к уравнению AᵀYᵀ = Bᵀ, решить его относительно Yᵀ преобразованиями строк (как в первом случае) и в конце снова транспонировать результат.
