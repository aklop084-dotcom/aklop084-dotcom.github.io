Билет 20. Применение ранга к исследованию систем линейных уравнений. Выражение общего решения однородной и неоднородной систем линейных уравнений через фундаментальную систему решений. Критерий совместности системы линейных уравнений в терминах ранга (теорема Кронекера-Капелли) и её следствие (условие единственности решения).

Применение ранга к исследованию систем линейных уравнений

Ранг матрицы играет ключевую роль в анализе систем линейных уравнений, так как он позволяет не только определить, есть ли у системы решения, но и описать их структуру. Нам нужно понятие ранга для построения полной теории разрешимости линейных систем, которая является фундаментом всей линейной алгебры.

Критерий совместности (Теорема Кронекера-Капелли)

Этот критерий дает ответ на самый первый вопрос при встрече с системой: существуют ли решения вообще?

📌 Объяснение словами

Система линейных уравнений имеет хотя бы одно решение тогда и только тогда, когда добавление столбца свободных членов к матрице коэффициентов не увеличивает её ранг. Это означает, что столбец свободных членов должен «вписываться» в структуру матрицы коэффициентов, являясь линейной комбинацией её столбцов.

📌 Теорема Кронекера-Капелли

Пусть дана система AX = b, где A — матрица коэффициентов размера m×n, а A* — расширенная матрица, полученная путем приписывания к A столбца свободных членов b. Система совместна (имеет решение) тогда и только тогда, когда справедливо равенство:

rank A = rank A*

Если ранг основной матрицы меньше ранга расширенной матрицы (rank A < rank A*), то система несовместна и решений не имеет.

Условие единственности решения (Следствие)

Если мы установили, что система совместна, следующим шагом необходимо понять, сколько у неё решений: одно или бесконечно много.

📌 Объяснение словами

Для того чтобы решение совместной системы было единственным, необходимо, чтобы ранг матрицы был максимально возможным и совпадал с числом неизвестных. В этом случае в системе нет «лишних» или свободных переменных, которые могли бы принимать произвольные значения.

📌 Следствие (Условие единственности)

Пусть система AX = b совместна и число неизвестных равно n.

  • Система имеет единственное решение (называется определенной) тогда и только тогда, когда rank A = n. В этом случае определитель матрицы (если она квадратная) отличен от нуля.
  • Система имеет бесконечно много решений (называется неопределенной) тогда и только тогда, когда rank A < n. Число свободных неизвестных в этом случае равно n - rank A.

Фундаментальная система решений и общее решение однородной системы

Для однородной системы уравнений (где все свободные члены равны нулю) решения всегда существуют (как минимум, тривиальное — нулевое). Нам нужна фундаментальная система решений для описания всего пространства решений как линейной оболочки нескольких векторов.

📌 Определение фундаментальной системы решений (ФСР)

Любой базис в подпространстве решений однородной системы уравнений AX = 0 называется фундаментальной системой решений. Если ранг матрицы A равен r, а число неизвестных — n, то фундаментальная система решений состоит из n - r векторов.

📌 Выражение общего решения однородной системы

Общее решение Xоднор представляется как произвольная линейная комбинация векторов фундаментальной системы решений:

Xоднор = c₁·X⁽¹⁾ + c₂·X⁽²⁾ + ⋯ + c_{n-r}·X⁽ⁿ⁻ʳ⁾

где X⁽¹⁾, …, X⁽ⁿ⁻ʳ⁾ — векторы фундаментальной системы решений, а c₁, …, c_{n-r} — произвольные числа (скаляры).

Общее решение неоднородной системы линейных уравнений

Решения неоднородной системы (AX = b, b ≠ 0) не образуют подпространства, но их структуру можно описать, опираясь на соответствующую однородную систему.

📌 Объяснение словами

Чтобы найти все решения неоднородной системы, достаточно найти одно любое её частное решение и прибавить к нему общее решение соответствующей однородной системы. Таким образом, неоднородность просто «сдвигает» всё пространство решений однородной системы на один конкретный вектор.

📌 Структура общего решения неоднородной системы

Пусть X₀ — любое частное решение неоднородной системы AX = b, а {X⁽¹⁾, …, X⁽ⁿ⁻ʳ⁾} — фундаментальная система решений соответствующей однородной системы AX = 0. Тогда общее решение неоднородной системы записывается в виде:

X = X₀ + c₁·X⁽¹⁾ + c₂·X⁽²⁾ + ⋯ + c_{n-r}·X⁽ⁿ⁻ʳ⁾

где cᵢ — произвольные коэффициенты.

Метод построения общего решения (пошагово)

Для практического применения теории ранга используется следующий алгоритм (на базе метода Гаусса):

📌 Алгоритм построения общего решения
  • Шаг 1. Приведение матрицы к ступенчатому виду: С помощью элементарных преобразований строк расширенную матрицу системы (A∣b) приводят к ступенчатому виду. На этом этапе по теореме Кронекера-Капелли проверяют совместность системы.
  • Шаг 2. Определение ранга и разделение переменных: Находят ранг r. Выделяют r базисных (главных) переменных и n - r свободных переменных.
  • Шаг 3. Нахождение частного решения: Придают всем свободным переменным значения 0 и вычисляют значения главных переменных из ступенчатой системы. Полученный вектор — это X₀.
  • Шаг 4. Построение фундаментальной системы решений: По очереди придают одной из свободных переменных значение 1, а остальным — 0 (при этом b в правой части уравнений заменяют на 0). Вычисляют значения главных переменных. Полученные n - r векторов составят фундаментальную систему решений.
  • Шаг 5. Запись общего решения: Комбинируют полученные результаты в итоговую формулу X = X₀ + Σ cᵢ·X⁽ⁱ⁾.
