Билет 18. Сумма и пересечение подпространств. Формула для размерности суммы двух подпространств (формула Грассмана). Прямая сумма подпространств. Критерий прямой суммы двух подпространств.

Пересечение и сумма подпространств

В линейной алгебре часто возникает необходимость комбинировать уже имеющиеся подпространства или находить их общие элементы. Нам это нужно для того, чтобы анализировать структуру сложных линейных пространств, разлагая их на более простые составляющие, например, при изучении собственных подпространств операторов.

📌 Пересечение подпространств

Словами: пересечением двух подпространств называется множество всех векторов, которые одновременно принадлежат и первому, и второму подпространству. Это своего рода «общая часть» двух структур.

Строгим математическим языком: пусть L₁ и L₂ — подпространства линейного пространства L. Пересечением L₁ ∩ L₂ называется множество:

L₁ ∩ L₂ = {x ∈ L | x ∈ L₁ и x ∈ L₂}

Пересечение любого количества подпространств само всегда является подпространством.

📌 Сумма подпространств

Важно понимать, что простое объединение подпространств (как множеств) обычно не является подпространством, так как сумма вектора из одного пространства и вектора из другого может вылететь за пределы обоих. Поэтому вводится понятие суммы: это множество всех векторов, которые можно представить как сумму двух слагаемых, где первое взято из первого подпространства, а второе — из второго.

Строгим математическим языком: суммой L₁ + L₂ подпространств L₁ и L₂ называется линейная оболочка их объединения L₁ ∪ L₂, что эквивалентно множеству:

L₁ + L₂ = {x₁ + x₂ | x₁ ∈ L₁, x₂ ∈ L₂}

Сумма подпространств является наименьшим подпространством, содержащим в себе и L₁, и L₂.

Формула размерности суммы (Формула Грассмана)

Размерность суммы подпространств не всегда равна сумме их размерностей. Если у пространств есть общие векторы (ненулевое пересечение), то при простом сложении размерностей эти общие направления будут посчитаны дважды. Чтобы получить верный результат, размерность пересечения нужно вычесть.

📌 Теорема (Формула Грассмана)

Для любых двух конечномерных подпространств L₁ и L₂ справедливо равенство:

dim(L₁ + L₂) = dim L₁ + dim L₂ - dim(L₁ ∩ L₂)

📐 Пример

Если в трехмерном пространстве взять две различные плоскости, проходящие через начало координат (размерность каждой равна 2), они обязательно пересекутся по прямой (размерность 1). Тогда размерность их суммы будет 2 + 2 - 1 = 3, что логично, так как две такие плоскости вместе «заметают» всё трехмерное пространство.

Прямая сумма подпространств

Прямая сумма — это идеальный случай сложения подпространств, когда они совершенно не «перекрываются» (их пересечение содержит только нулевой вектор). В этом случае каждый вектор из получившейся суммы можно собрать из векторов-слагаемых единственным возможным способом. Нам нужна прямая сумма для однозначного разложения векторов по базисам, согласованным с этими подпространствами.

📌 Определение прямой суммы

Сумма подпространств L₁ + L₂ называется прямой суммой (обозначается L₁ ⊕ L₂), если любой вектор x из этой суммы допускает единственное представление в виде x = x₁ + x₂, где x₁ ∈ L₁ и x₂ ∈ L₂.

Критерий прямой суммы двух подпространств

Чтобы проверить, является ли сумма двух подпространств прямой, не обязательно проверять единственность разложения для каждого вектора. Достаточно убедиться, что у этих пространств нет общих ненулевых векторов.

📌 Утверждение (Критерий прямой суммы)

Сумма двух подпространств L₁ и L₂ является прямой тогда и только тогда, когда их пересечение состоит только из нулевого вектора:

L₁ ⊕ L₂   ⟺   L₁ ∩ L₂ = {0}
📐 Доказательство:

Необходимость (Если сумма прямая ⇒ пересечение нулевое): Пусть сумма прямая. Предположим от противного, что в пересечении есть вектор y ≠ 0. Тогда вектор 0 можно представить двумя способами: как 0 + 0 (где первое 0 ∈ L₁, второе 0 ∈ L₂) и как y + (-y) (так как y в пересечении, то y ∈ L₁ и -y ∈ L₂). Это противоречит единственности разложения в прямой сумме. Значит, y может быть только нулем.

Достаточность (Если пересечение нулевое ⇒ сумма прямая): Пусть L₁ ∩ L₂ = {0}. Предположим, что какой-то вектор x можно разложить двумя способами: x = x₁ + x₂ и x = x₁′ + x₂′, где x₁, x₁′ ∈ L₁ и x₂, x₂′ ∈ L₂. Тогда x₁ + x₂ = x₁′ + x₂′ ⇒ x₁ - x₁′ = x₂′ - x₂. Вектор в левой части равенства принадлежит L₁, а в правой — L₂. Значит, этот вектор принадлежит пересечению. Так как пересечение нулевое, то x₁ - x₁′ = 0 и x₂′ - x₂ = 0, откуда x₁ = x₁′ и x₂ = x₂′. Разложение единственно, сумма прямая.

📐 Следствие из формулы Грассмана:

Из формулы Грассмана также следует, что для прямой суммы размерность суммы равна сумме размерностей слагаемых, так как dim(L₁ ∩ L₂) = 0:

dim(L₁ ⊕ L₂) = dim L₁ + dim L₂
📐 Линейная комбинация

Линейная комбинация — это математическое выражение, полученное путем умножения набора элементов (векторов, функций, чисел) на числовые коэффициенты с последующим их сложением.

α·v₁ + β·v₂ + γ·v₃ + … + ω·vₙ

где:

  • v₁, v₂, v₃, …, vₙ — элементы (векторы, неизвестные, функции), которые комбинируются;
  • α, β, γ, …, ω — коэффициенты (скаляры), на которые умножаются элементы.

В контексте суммы подпространств, любой вектор из L₁ + L₂ представляется в виде линейной комбинации x = x₁ + x₂, где x₁ и x₂ — векторы из соответствующих подпространств. В случае прямой суммы это представление единственно.
