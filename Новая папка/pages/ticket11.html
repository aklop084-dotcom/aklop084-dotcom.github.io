Билет 11. Смешанное произведение трех векторов, его свойства, выражение в координатах. Объем ориентированного параллелепипеда. Критерий компланарности (линейной зависимости) трех векторов в терминах смешанного произведения.

Смешанное произведение трех векторов

Смешанным произведением трех векторов a, b и c называется число (скаляр), которое получается, если вектор a умножить скалярно на векторное произведение векторов b и c. Нам нужно это понятие для решения объемных задач аналитической геометрии, например, для вычисления объемов многогранников или проверки того, лежат ли векторы в одной плоскости.

📌 Определение смешанного произведения

На строгом математическом языке определение формулируется так: Смешанным произведением трех векторов a, b, c называется число, обозначаемое (a, b, c) или abc, равное:

(a, b, c) = (a, [b, c])

где (·, ·) — скалярное произведение, а [·, ·] — векторное произведение.

Геометрический смысл и объем параллелепипеда

Геометрически смешанное произведение тесно связано с понятием объема.

📌 Геометрический смысл

Словами: Абсолютная величина (модуль) смешанного произведения трех векторов равна объему параллелепипеда, построенного на этих векторах как на ребрах. Знак произведения зависит от ориентации тройки векторов: если тройка правая, произведение положительно; если левая — отрицательно.

На строгом математическом языке: Пусть V — объем параллелепипеда, построенного на векторах a, b, c. Тогда:

V = |(a, b, c)|

Понятие ориентированного объема S±(a, b, c) совпадает со значением смешанного произведения:

S±(a, b, c) = (a, b, c)

Это значение равно произведению площади основания параллелепипеда (модуля векторного произведения [b, c]) на высоту, опущенную из конца вектора a на плоскость векторов b и c.

Свойства смешанного произведения

Смешанное произведение обладает следующими важными свойствами:

❶ Циклическая перестановка
Смешанное произведение не меняется при циклической перестановке множителей.
(a, b, c) = (b, c, a) = (c, a, b)
❷ Антикоммутативность
При перемене мест любых двух соседних векторов смешанное произведение меняет знак на противоположный.
(a, b, c) = -(b, a, c)
❸ Линейность
Операция линейна по каждому из своих множителей (свойства дистрибутивности и возможности вынесения скалярного множителя).
❹ Связь операций
В смешанном произведении знаки скалярного и векторного умножения можно менять местами.
(a, [b, c]) = ([a, b], c)

Выражение в координатах

Если в пространстве задан правый ортонормированный базис, и векторы заданы своими координатами a = (a₁, a₂, a₃), b = (b₁, b₂, b₃) и c = (c₁, c₂, c₃), то смешанное произведение вычисляется через определитель.

📌 Пошаговый метод вычисления
  • Составить матрицу третьего порядка.
  • В первую строку записать координаты первого вектора a.
  • Во вторую строку записать координаты второго вектора b.
  • В третью строку записать координаты третьего вектора c.
  • Вычислить определитель этой матрицы.
(a, b, c) = ⎡ a₁ a₂ a₃ ⎤ ⎢ b₁ b₂ b₃ ⎥ ⎣ c₁ c₂ c₃ ⎦

Доказательство этого факта основано на разложении векторного произведения по базису и последующем применении формулы скалярного произведения в координатах.

Критерий компланарности трех векторов

Векторы называются компланарными, если они параллельны одной и той же плоскости (или лежат в ней). Нам нужно знать условие компланарности, например, для проверки того, образуют ли три вектора базис в трехмерном пространстве (если они не компланарны, то образуют).

📌 Критерий компланарности

Утверждение: Три вектора a, b, c компланарны (линейно зависимы) тогда и только тогда, когда их смешанное произведение равно нулю.

(a, b, c) = 0   ⟺   a, b, c компланарны

Словами: Это утверждение очевидно из геометрического смысла. Если векторы лежат в одной плоскости, то параллелепипед, построенный на них, «сплющивается», его высота становится равной нулю, и, следовательно, объем равен нулю.

📐 Пример

Нам нужно проверить, лежат ли векторы a = (1, 2, 3), b = (1, 1, 1) и c = (0, 1, 2) в одной плоскости.

Решение: Составляем определитель:

| 1 2 3 | | 1 1 1 | = 1·(1·2 - 1·1) - 2·(1·2 - 1·0) + 3·(1·1 - 1·0) = | 0 1 2 |
= 1·(2 - 1) - 2·(2 - 0) + 3·(1 - 0) = 1·1 - 2·2 + 3·1 = 1 - 4 + 3 = 0

Так как смешанное произведение равно нулю, векторы компланарны.
