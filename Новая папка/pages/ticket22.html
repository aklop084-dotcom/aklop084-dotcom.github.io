Билет 22. Собственный вектор и собственное значение линейного оператора. Линейная независимость собственных векторов, отвечающих различным собственным значениям. Отыскание собственных векторов с помощью матрицы. Характеристическое уравнение и характеристический многочлен матрицы линейного оператора. Диагонализируемость матрицы линейного преобразования (необходимое и достаточное условие – существование базиса из собственных векторов; достаточное условие – наличие n различных собственных значений).

Этот билет посвящен изучению «особых» направлений линейного оператора — таких векторов, которые при преобразовании лишь растягиваются или сжимаются, не меняя своей ориентации. Эти понятия позволяют максимально упростить матрицу оператора, приведя её к диагональному виду.

1. Собственный вектор и собственное значение линейного оператора

📌 Словесное описание

Представьте, что линейный оператор действует на пространство как некая деформация. В большинстве случаев векторы после применения оператора поворачиваются и меняют длину. Однако для многих операторов существуют такие ненулевые векторы, которые после преобразования остаются на той же самой прямой, на которой они лежали изначально. Оператор лишь умножает такой вектор на некоторое число (растягивает, сжимает или отражает). Этот вектор называется собственным, а число, показывающее степень его растяжения, — собственным значением.

📌 Строгий математический язык

Пусть A: L → L — линейный оператор в линейном пространстве L. Ненулевой вектор x ∈ L называется собственным вектором линейного оператора A, если найдется такое число λ, что выполняется равенство:

A(x) = λx

Число λ при этом называется собственным значением (или собственным числом) оператора A, соответствующим вектору x.

2. Характеристическое уравнение и характеристический многочлен

📌 Словесное описание

Нам нужно найти собственные значения для того, чтобы в дальнейшем вычислить сами собственные векторы. Для этого мы переходим к матричной форме. Условие A(x) = λx можно переписать как (A - λE)x = 0. Это однородная система уравнений, и чтобы у неё были ненулевые решения (наши векторы x), определитель матрицы этой системы должен быть равен нулю. Так мы получаем уравнение, из которого находятся все возможные значения λ.

📌 Строгий математический язык

Пусть оператор A в некотором базисе имеет матрицу A. Определитель матрицы (A - λE), где E — единичная матрица, является многочленом относительно λ степени n (где n = dim L):

p(λ) = det(A - λE)

Этот многочлен называется характеристическим многочленом оператора A.

Уравнение

det(A - λE) = 0

называется характеристическим уравнением оператора A. Корни этого уравнения являются собственными значениями оператора.

3. Отыскание собственных векторов с помощью матрицы

📌 Метод отыскания по шагам
  • Шаг 1. Составление матрицы: Записать матрицу оператора A в произвольном (обычно стандартном) базисе.
  • Шаг 2. Поиск собственных значений: Составить характеристическое уравнение det(A - λE) = 0 и найти все его корни λ₁, λ₂, …, λₙ.
  • Шаг 3. Поиск векторов для каждого λ: Для каждого найденного корня λᵢ подставить его в систему линейных однородных уравнений: (A - λᵢE)ξ = 0, где ξ — столбец координат искомого собственного вектора.
  • Шаг 4. Нахождение фундаментальной системы решений: Решить полученную систему. Любое ненулевое решение (или их линейная комбинация) будет являться собственным вектором, отвечающим данному λᵢ.

📐 Пример

Если матрица A = ⎡ 1 0 ⎤, то характеристическое уравнение (1 - λ)(2 - λ) = 0 дает корни λ₁ = 1 и λ₂ = 2.

      ⎣ 0 2 ⎦

Подставляя λ₁ = 1, получаем систему: 0·x₁ + 0·x₂ = 0 и (2 - 1)x₂ = 0, откуда x₂ = 0, а x₁ — любое. Собственный вектор — (1, 0).

4. Линейная независимость собственных векторов

📌 Словесное описание

Если у нас есть несколько собственных векторов, которые «растягиваются» по-разному (то есть отвечают разным собственным значениям), то они гарантированно не лежат на одной прямой и не могут быть выражены друг через друга. Они указывают на принципиально разные «чистые» направления действия оператора.

📌 Теорема

Собственные векторы линейного оператора, отвечающие попарно различным собственным значениям, линейно независимы.

📐 Доказательство (словами): Предположим обратное: что между такими векторами есть зависимость. Если мы применим к этой зависимости оператор, то каждый вектор умножится на «своё» число λ. Вычитая исходную зависимость, умноженную на одно из λ, мы будем постепенно исключать векторы, пока не придем к противоречию (например, что ненулевой вектор равен нулю).

5. Диагонализируемость матрицы линейного оператора

📌 Словесное описание

Нам нужно иметь возможность записывать оператор в максимально простом виде для быстрого возведения матрицы в степень или анализа динамических систем. Самый простой вид матрицы — диагональный, когда ненулевые элементы стоят только на главной диагонали.

📌 Необходимое и достаточное условие диагонализируемости

Матрица линейного оператора A в некотором базисе является диагональной тогда и только тогда, когда этот базис состоит из собственных векторов этого оператора.

В этом случае на главной диагонали матрицы будут стоять соответствующие собственные значения.

📌 Достаточное условие диагонализируемости

Если все корни характеристического многочлена оператора, действующего в n-мерном пространстве, различны (то есть имеется n различных собственных значений), то такой оператор всегда можно привести к диагональному виду.

Логика здесь проста: раз есть n различных значений, то есть n линейно независимых собственных векторов, которые в n-мерном пространстве автоматически образуют базис.
